# FinalProject
WanderingInTheWoods

Description:

Wandering in the Woods Game: People are “lost in the woods” where the woods are represented by a rectangular grid. The woods are dense, and the people can’t see or hear each other until they are in the same cell of the grid.

To run this program locally:

1.
  Download zip file
  Extract the files 
  -OR-
  Clone the repository:

    git clone https://github.com/rns-code/FinalProject.git

2.
On Windows:
  Open the directory and run WanderingInTheWoods.bat
  If prompted, select Run

On Mac/Linux:
  Run using Wine
  In command line:
  
    wine cmd.exe /c WanderingInTheWoods.bat
